5. 说明

  • 将 cx_xxx_replace_with_your_key 替换成你自己的 API Key。
  • 直连端口模式下使用 /anthropic、/v1、/compat/v1。
  • nginx 前缀模式下使用 /codex/anthropic、/codex/v1、/codex/compat/v1。
  • Claude Code 原生入口优先使用 Anthropic 兼容入口，不要优先走 chat/completions。
  • 如果你走 OAuth loopback 模式，浏览器会跳到 http://localhost:1455/auth/callback，这是正常的本地回调。
